== Description ==

Our WordPress Plugin makes it simple to install our software into your wordpress site.  Simply install the plugin and update your configuration with the project ID supplied when setting up your project.

<a title="RevisionLab | Software to Make Web Development Easier" href="http://revisionlab.com/">RevisonLab - Software for Web Development Feedback Management</a>.

RevisionLab is a software to help web development companies more easily manage their feedback and revision process.  
It easily integrates into any development website to be used quickly on any project.
